# Two-Person, Zero-Sum, Finite Matrix Games

## 簡介

Player1和Player2各有`n`, `m`個選擇，
有一個矩陣A(i, j)描述著Player1和Player2做出選擇
（Player1選擇`i`, Player2選擇`j`）後，
Player1會獲得多少錢

像是假如矩陣長這樣：

那當Player1選1且Player2選1的時候，Player2要給Player1`1`塊錢
